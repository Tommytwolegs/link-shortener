5 — Address bar, popup, shortcut, omnibox (no links needed)

  1. Address bar: open https://www.youtube.com/watch?v=dQw4w9WgXcQ&si=AbCdEf123 — the si= should vanish from the address bar in place, no reload.
  2. Popup: on that page, click the toolbar icon — preview should show the clean URL and “Already clean” (you arrived pre-cleaned). Click Copy clean URL → “Copied ✓”.
  3. Filter box: type ebay — one row remains, its group auto-opens. Press Esc — full list and your open/closed groups come back. Type zzz — “No sites match”.
  4. Shortcut: press Ctrl+Shift+L on any normal page, paste — clean URL. The popup’s tip line should show the shortcut Chrome actually assigned (or “not set” if another extension owns it — check chrome://extensions/shortcuts).
  5. Omnibox: in the address bar type clean, press Tab, type hello — hint says “No URL detected yet”. Now paste https://www.youtube.com/watch?v=dQw4w9WgXcQ&si=AbCdEf123 — suggestion previews the clean form; Enter navigates to it.
  6. Restricted pages: on chrome://extensions, right-click — the “Copy clean URL” items should NOT appear at all (that’s intentional).

This page is dev-only; it is not shipped in the store packages. Crafted links point at real sites but were never served by the wrappers they imitate — that's fine, the unwrapper never touches the network.
